Update vom CBA Modul auf das neue Amazon Pay Modul
Sofern Sie noch ein ganz altes CBA Modul haben und auf das neue Amazon Pay Modul wechseln möchten, müssen Sie zunächst im Administrationsbereich, unter
Zahlungsmodule, das alte Amazon Payments Modul deinstallieren. Danach müssen Sie alle getätigten Änderungen im Code rückgängig machen und die
hinzugefügten Verzeichnisse und Dateien allesamt löschen. Anschließend können Sie das neue Amazon Pay Modul installieren.

Update von einer älteren Version des API Moduls
Sofern Sie noch ein älteres API Modul haben und auf das neue Amazon Pay Modul wechseln möchten, müssen Sie zunächst im Administrationsbereich, unter
Zahlungsmodule, das alte Amazon Payments Modul deinstallieren. Danach müssen Sie alle getätigten Änderungen im Code rückgängig machen und die
hinzugefügten Verzeichnisse und Dateien allesamt löschen. Anschließend können Sie das neue Amazon Pay Modul installieren.

Wichtiger Hinweis!
Sofern Sie ein anderes Template nutzen, als das Standardtemplate, müssen Sie darauf achten, dass alle Änderungen im alternativen Templateverzeichnis erfolgen
und dass die neuen Dateien, die das Template betreffen, in das Verzechniss des alternativen Templates kopiert werden.
